edamam food recipe

This was a project I started back in June 2022

The system makes get requests to the edamam end point.

The result would be a json object sent to the clients brower and mapped to the console by the help of react.

Customers has the ability to view the recipes and filter the searchs

I worked on this system for about 2 weeks.

Technologies:
  • - Node js
  • - Express REST Framework
  • - React JS
  • - Heroku
  • - Edamam API API
